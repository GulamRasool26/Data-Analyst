# Data Analyst Learning Workspace

This folder is a personal learning and practice workspace for data analysis, covering Python, SQL, Excel, Power BI, web scraping, and data projects.

## What’s included

- Python
  - Basics: variables, data types, operators, loops, functions, conditionals
  - Pandas: reading data, cleaning, filtering, grouping, merging, and visualization
  - Web scraping: requests, BeautifulSoup, and HTML parsing

- SQL
  - Beginner topics: SELECT, WHERE, GROUP BY, ORDER BY, LIMIT, aliases
  - Intermediate topics: joins, subqueries, CASE statements, unions, window functions
  - Advanced topics: CTEs, stored procedures, temporary tables, triggers
  - Project work with data cleaning and exploratory analysis

- Excel
  - Basic, advanced, cleaning, and project-based exercises

- Power BI
  - Basic and advanced dashboards, reports, and projects

- Web Scraping
  - Sample projects using Python to collect and analyze data from websites

## Folder structure

- AmazonWebScraping/: web scraping notebooks and datasets
- Excel/: Excel practice files and projects
- PowerBI/: Power BI learning materials and projects
- Python/: Python notebooks and datasets
- SQL/: SQL scripts, datasets, and project files

## Goals

This workspace is intended for:
- learning core data analysis skills
- practicing real-world data tasks
- building portfolio projects
- improving data cleaning, analysis, and visualization abilities

## Suggested next steps

1. Start with the Python basics notebooks.
2. Move on to SQL fundamentals and practice queries.
3. Explore Pandas for data manipulation and analysis.
4. Build dashboards in Power BI and Excel projects.
